# Bamazon
  Part functional e-commerce website made to learn Django.

# Features
  Login & Register <br>
  Adding & Removing to/from cart
 
# Requirements
  Django <br>
  Crisp forms
  
# How to
  1. Clone repository
  2. Migrate DB (manage.py migrate)
  3. Run server (manage.py runserver)
